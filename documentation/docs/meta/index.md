# Meta Tables

| Name | Summary |
|------|---------|
| [Character Meta](./character.md) | The character meta table provides comprehensive functionality for managing character data, attributes, and operations in the Lilia framework. |
| [Entity Meta](./entity.md) | The entity meta table provides comprehensive functionality for extending Garry's Mod entities with Lilia-specific features and operations. |
| [Inventory Meta](./inventory.md) | The inventory meta table provides comprehensive functionality for managing inventory data, item storage, and inventory operations in the Lilia framework. |
| [Item Meta](./item.md) | The item meta table provides comprehensive functionality for managing item data, properties, and operations in the Lilia framework. |
| [Panel Meta](./panel.md) | The panel meta table provides comprehensive functionality for managing VGUI panels, UI interactions, and panel operations in the Lilia framework. |
| [Player Meta](./player.md) | The player meta table provides comprehensive functionality for managing player data, interactions, and operations in the Lilia framework. |
