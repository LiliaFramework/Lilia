# Advanced Duplicator 2 Compatibility

Provides compatibility and security measures for the Advanced Duplicator 2 addon within the Lilia framework.

---

Overview

The Advanced Duplicator 2 compatibility module ensures safe and controlled usage of the Advanced Duplicator 2 addon. It implements security measures to prevent crash exploits and unauthorized duplication.
The module operates on the server side to validate duplication attempts, checking for oversized models and entities marked as non-duplicable, with enhanced support for AdvDupe2's specific data structures.
It includes logging functionality to track potential security violations and crash attempts, along with integration with AdvDupe2's notification system.
The module provides enhanced duplication controls and integrates with the framework's permission system for advanced duplication management.
