**Name:**
Slot Machine

**Version:**
1.16

**Description:**

Introduces a slot machine minigame using a workshop model. Players can gamble for configurable payouts with sounds and animations.

<h2 style="text-align:center; font-size:1.5rem; font-weight:bold;">Features</h2>

- Adds a slot machine minigame
- Adds a workshop model for the machine
- Adds handling of payouts to winners
- Adds customizable payout odds
- Adds sound and animation effects
